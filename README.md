# 敏捷攀樹手冊
「敏捷攀樹手冊」(Agile Tree Climbing Manual)是以「本然」概念出發撰寫此手冊，除了專業的「攀樹技術」外也會融合「本然教育」的教學方式、理念與目標，以此作為團隊的參考手冊。

同時團隊的運作模式會朝著「[敏捷活動規劃宣言](https://github.com)」()精神運作，因此敏捷攀樹手冊會是在攀樹社群的回饋下不斷的更新，因此手冊會將「攀樹技術」、「敏捷活動規劃宣言」與「本然教育」做融合。
